Privacy Policy

// Last updated: May 2026

1. Overview

This website (kiwiprojekt.pl) is a static landing page displaying public open-source projects hosted by the kiwiprojekt GitHub organization. We value your privacy and aim to keep data collection to an absolute minimum.

2. Data Collection & Processing

As the maintainers of this website, we do not directly collect, store, or process any personal data, nor do we run cookies, trackers, or marketing analytics.

However, because the website is hosted on the internet, connection data must be processed technically by hosting and utility providers:

  • Hosting (GitHub Pages): This website is hosted using GitHub Pages (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA). When you visit the site, GitHub automatically collects and processes connection metadata (including your IP address, browser type, referrer, and timestamps) in server logs to ensure secure and reliable delivery of the pages. For details, please review the GitHub Privacy Statement.
  • External Libraries (unpkg.com): To run the interactive elements of this static page, scripts (React and Babel Standalone) are retrieved dynamically from unpkg.com (mPn Trend S.L.). When your browser requests these resources, your IP address is visible to the service.

3. Self-Hosted Fonts

To prevent tracking by third-party search engines and CDNs, all font assets (including Fraunces, IBM Plex Sans, and JetBrains Mono) are **fully self-hosted** on our own server space. No connections or requests are made to external Google Fonts CDNs.

4. Cookies

This website does not use cookies, web beacons, pixels, or other tracking technologies. There are no tracking mechanisms that store data on your device.

5. Your Rights

Under the GDPR, you have the right to access, rectify, or erase any personal data processed in connection with this site. Since we do not store any personal data, please contact the hosting provider (GitHub) for inquiries regarding server logs.

6. Contact

If you have any questions about this Privacy Policy, please open an issue or reach out via our public GitHub organization profile: github.com/kiwiprojekt.
